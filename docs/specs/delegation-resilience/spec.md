## Problem

Six residuals from live delegated-run experience (downstream proposal paperclip+hermes-v1 2026-06-12, each evidenced in run records): (1) delegated sessions die mid-work (provider limits, restarts) and recovery worked only because deploys/tests happened to leave traces — progress artifacts are end-loaded, not written as facts occur; (2) at an unbounded wait (human approval gate) the turn-completion rule bans the false "I'll resume" but prescribes no honest exit; (3) delegated returns re-paste artifact bodies (~hundreds of duplicated lines/slice) — pointers-over-reprints exists as principle but has no return SHAPE; (4) reused procedural templates propagate superseded procedures; (5) `docs/tmp/handoff.md` doubles as operator-triggered restart artifact AND autonomous run ledger — the dual trigger confused the operator; (6) the token-waste audit is per-session by construction — cross-session recurring costs are invisible.

## Decisions (locked; plan-review deltas marked PR:F#)
