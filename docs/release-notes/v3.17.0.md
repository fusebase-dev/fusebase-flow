# Fusebase Flow v3.17.0 — context-floor reduction (−~30% always-on session cost)

**Release date:** 2026-06-10
**Previous version:** 3.16.4

## Summary

The structural half of the framework-wide efficiency audit (mechanical repairs shipped as v3.16.4). Every agent session pays an always-on floor before any work happens — measured at ~34.5k tokens (Claude Code) / ~27.9k (Codex) at v3.16.4. This release cuts it **~8k tokens per session, every session, every consumer repo** (PO −8.0k · AI-Dev −7.9k · Deploy −8.5k at bytes/4). Built by an independent implementer agent against a locked spec; an independent reviewer attested **per-rule** that no FR semantics changed.
